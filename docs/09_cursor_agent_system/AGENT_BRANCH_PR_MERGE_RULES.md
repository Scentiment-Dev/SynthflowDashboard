# Agent Branch, PR, and Merge Rules

- Work from short-lived branches.
- Use `wave-<number>/<agent>/<task>` branch names.
- Agents may open PRs.
- Agents may not merge their own PRs.
- Agent C performs no-drift review on functional changes.
- AI PM accepts or rejects the PR.
- Direct-to-main changes are forbidden after Wave 8.
