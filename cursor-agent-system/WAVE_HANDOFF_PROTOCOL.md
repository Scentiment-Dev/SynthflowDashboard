# Wave Handoff Protocol

At the end of every wave, AI PM must update:

- `HANDOFF_README.md`
- `REHYDRATION_PROMPT.md`
- `project-management/wave_status.md`
- `project-management/wave_schedule.md`
- `project-management/wave_schedule.json`
- `project-management/wave_<number>_completion_report.md`
- `repo_manifest.json`

The final answer for each wave must include what completed, what changed, validation performed, what was not run, download links, and next scheduled wave.
