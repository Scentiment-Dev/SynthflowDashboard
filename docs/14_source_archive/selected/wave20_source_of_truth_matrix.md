# Wave 2 Source-of-Truth Matrix

| Domain | Source of Truth | Notes |
|---|---|---|
| Subscription state | Stay.ai | Final authority |
| Subscription action outcome | Stay.ai | Required for completion |
| Save/retained outcome | Stay.ai or approved official path | Acceptance alone is not save |
| Cancellation outcome | Stay.ai or approved official path | Request alone is not cancellation |
| Call intent and branch | Synthflow | Journey source, not outcome source |
| Handoff sent | Synthflow/email/SMS provider | Link sent only |
| Handoff/portal completion | Portal/Stay.ai completion source | Required for portal success |
| Order/product/customer context | Shopify | Context only |
| Escalation outcome | RingCX/live-agent where integrated | Escalation/follow-up source |
